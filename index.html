Codr is a collaborative pastebin for code. Like Google Docs, every edit and cursor move is instantly synced to all users. For web developers, Codr also offers shared, live HTML/CSS/JS rendering. Codr is lightweight and focused: it's not an IDE and does not aim to be. As a result, Codr is delightfully easy to use and has unmatched performance. The initial version of Codr is already built, but we need your help to launch!
